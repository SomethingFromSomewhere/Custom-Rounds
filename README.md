# [CORE] Custom Rounds

This plugin provides features for building custom rounds.

# Config example:
```
"CustomRounds"
{
    "AK47"
    {
        "weapon"        "weapon_ak47"
        "ammo"            "2"
        "no_buy"        "1"
    }
 
    "AWP"
    {
        "ammo"            "1"
        "weapon"        "weapon_awp"
        "no_zoom"        "1"
        "no_knife"        "1"
        "no_buy"        "1"
    }
 
    "SpeedUP"
    {
        "gravity"            "0.5"
        "speed"                "2"
        "invisibility"        "155"
        "hp"                "50"
        "armor"                "50"
        "cmd"                "sm_drug @all"
    }
}
```
